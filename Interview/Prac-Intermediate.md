
## Level 2: Intermediate Git Practical Questions

1. **Create a new branch called `feature-login` and switch to it.**

2. **Create a new file named `login.txt`, add some content to it, and commit the changes in the `feature-login` branch.**

3. **Switch back to the `main` branch.**

4. **Merge the `feature-login` branch into the `main` branch.**

5. **View the list of all branches available in the repository.**

6. **Delete the branch `feature-login` after merging it with the main branch.**

7. **Make some changes to a file and temporarily save those changes using Git stash.**

8. **View the list of stashed changes and apply the latest stash back to the working directory.**

9. **Add a remote repository to your local Git project.**

10. **Push the `main` branch of your local repository to the remote repository.**

11. **Compare the difference between two commits in the repository.**

12. **View the commit history in a compact one-line format.**
